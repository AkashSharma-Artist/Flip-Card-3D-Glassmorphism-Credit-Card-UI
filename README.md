# Flip Card – 3D Glassmorphism Credit Card UI

![](https://github.com/AkashSharma-Artist/Flip-Card-3D-Glassmorphism-Credit-Card-UI/blob/main/%7B3C4B0D09-6964-4FA2-B929-A52C6E5B65CA%7D.png)

Flip Card is a front-end UI component that recreates a realistic credit 
card with a smooth 3D flip animation on hover. Built entirely with HTML 
and CSS3, it uses `perspective` and `transform-style: preserve-3d` to 
create genuine 3D flipping behavior, combined with a glassmorphism 
(frosted glass) design aesthetic.

## Features
- Realistic 3D card flip animation on hover (front ↔ back)
- Glassmorphism design — blurred backdrop, translucent card surface
- Front face: card logo, chip, card number, holder name, expiry date
- Back face: magnetic strip, signature panel, customer service details
- Decorative gradient orbs in the background for visual depth
- Fully responsive across desktop, tablet, and mobile breakpoints
- Zero JavaScript — pure CSS 3D transforms

## Tech Stack
- HTML5
- CSS3 (3D transforms, perspective, backdrop-filter, media queries)
- Google Fonts (Poppins, Merriweather)
